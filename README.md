# Care Voice AI

An AI-assisted voice agent for healthcare and clinical appointments.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:8080/`.

## Features
- Voice-assisted appointment booking
- Healthcare campaign management
- Integration with clinical backend services
